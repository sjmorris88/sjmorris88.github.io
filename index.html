Height Maps
mixed with 2D assets

This project was very unusual but very rewarding. This is an image of a detailed video game map from Red Dead Redemption 2. The benefits of using a 2D map which has the contour details provided is that you can digitally manipulate it to become a 3d bump map. I went half and half so the mountains look like they are coming out of the map.
